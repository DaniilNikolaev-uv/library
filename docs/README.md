# Документация проекта Archerion

Этот раздел содержит рабочие документы по проекту библиотечной системы `Archerion`.

## Содержание

- `Информация о проекте/Archerion.md` — краткий паспорт проекта: цель, стек, ключевые фичи, дорожная карта.
- `Информация о проекте/Полный_план.md` — подробное описание архитектуры, модели данных и функциональных модулей backend/frontend.
- `Информация о проекте/Apps Archerion.md` — декомпозиция backend на приложения (доменные модули).
- `Информация о проекте/Идеи для усиления проекта.md` — список витринных и защитных фич для усиления проекта.
- `Информация о проекте/Происхождение названия.md` — этимология названия проекта.

## Как использовать

- Для общего понимания проекта начните с `Archerion.md`.
- Для проектирования и реализации API используйте `Полный_план.md` как основной источник.
- Для структуры Django-приложений сверяйтесь с `Apps Archerion.md`.
- Для приоритизации следующих продуктовых улучшений используйте `Идеи для усиления проекта.md`.
