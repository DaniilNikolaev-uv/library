## Описание

`Archerion` — курсовой проект системы учета библиотечного фонда и выданных книг.
Система проектируется как веб-приложение с разделением на backend API и frontend-интерфейс для читателей и сотрудников.

## Основные фичи

- Каталог книг с фильтрацией по автору, жанру, году, ISBN и наличию.
- Учет физических экземпляров книги (`BookCopy`) со статусами и локациями.
- Выдача, возврат, продление и контроль просрочек (`Loan`).
- Бронирование книг/экземпляров и управление очередью (`Reservation`).
- Расчет и учет штрафов по настраиваемой политике (`Fine`, `FinePolicy`).
- Ролевой доступ: администратор, библиотекарь, читатель.
- Отчеты и аудит ключевых действий в системе.

## Технологии

- Языки: Python, TypeScript
- Фреймворки: Django REST Framework + Next.js
- БД: PostgreSQL
- Аутентификация: JWT (`djangorestframework-simplejwt`)
- Инструменты: Git, Docker, Docker Compose, Swagger/OpenAPI

## Архитектура

- Backend (Django/DRF): `accounts`, `catalog`, `inventory`, `circulation`, `reservations`, `fines`, `reports`, `audit`, `swagger`, `storage`.
- Frontend (Next.js App Router): публичный каталог, кабинет читателя, интерфейс библиотекаря, интерфейс администратора.
- Сервисный слой для бизнес-правил (выдача, возврат, продления, штрафы).
- REST API между frontend и backend.
- Отдельный сервис `healthcheck` показывает доступность backend, frontend, PostgreSQL, MinIO и pgAdmin.
